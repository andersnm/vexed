# Vexed Homepage

The official homepage for the Vexed Configuration Language, hosted at [https://andersnm.github.io/vexed](https://andersnm.github.io/vexed).

## Features

- **Landing Page**: Introduction to Vexed with features and examples
- **Language Reference**: Comprehensive documentation of all Vexed language features
- **Online Demo**: Interactive code editor to try Vexed in your browser

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Tech Stack

- **Vite**: Build tool and dev server
- **TypeScript**: Type-safe development
- **Tailwind CSS**: Utility-first CSS framework
- **CodeMirror**: Code editor for the online demo
- **Marked**: Markdown rendering for the language reference
- **Vexed**: The core Vexed library
- **Vexed-Browser**: Browser-specific Vexed runtime

## Deployment

The homepage is automatically deployed to GitHub Pages when changes are pushed to the main branch.
